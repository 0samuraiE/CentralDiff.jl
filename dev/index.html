CentralDiff.AbstractAxis — Type
AbstractAxis

An abstract type representing an axis. It serves as the base type for XAxis, YAxis, and ZAxis.

CentralDiff.dI — Type
dI{T<:AbstractAxis}(axis, stride)

Represent a displacement along a specific axis.

CentralDiff.TaylorMatrix — Method
TaylorMatrix(samples)

Construct a Taylor matrix from the given sample points. Returns a matrix where element (i,j) is i^j / j!.

Example

julia> A = TaylorMatrix(-3//2:3//2)
4×4 Matrix{Rational{Int64}}:
 1  -3//2  9//8  -9//16
 1  -1//2  1//8  -1//48
 1   1//2  1//8   1//48
 1   3//2  9//8   9//16

julia> iA = inv(A)
4×4 Matrix{Rational{Int64}}:
 -1//16   9//16   9//16  -1//16
  1//24  -9//8    9//8   -1//24
  1//2   -1//2   -1//2    1//2
  -1       3      -3       1
